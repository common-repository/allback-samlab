This plugin makes a simple widget for callback on your website.
	
== Description ==

Плагин предназначен для легкого создания виджета контактной формы для Вашео сайта.
Этот виджет будет выполнять функцию "обратного звонка". То есть на почту, указанную в настройках, Вам будет приходить номер телефона, который указал клиент в виджете на Вашем сайте.

== Installation ==

Просто установите и активируетй плагин, и на Вашем сайте появится виджет обратного звонка Bazz CallBack widget
